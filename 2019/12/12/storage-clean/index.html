Mac清内存

前几天发现内存老不够用，今天索性清一下。

先点屏幕左上角About this Mac调出存储查询，看看哪个部分占内存最多。如果Document占的多，可以直接通过内置的management进行修改。这里主要介绍Others的清理方法。

首先必须明确Others里面都包含哪些内容。
所有的文档，包括 .pdf .doc .psd所有的压缩文件和磁盘映像，包括 .zip .dmg .iso所有的用户数据所有的系统文件，包括 System 文件夹 和 cache 文件夹所有的 Spotlight 不认识的文件，包括 .exe 在内的对 OS X 来说奇奇怪怪的后缀名所有的不是 OS X 标准格式的视频、音频文件所有的应用插件和扩展
大概就是各种平时用到的用不到的文件了，怪不得占内存这么多。。

OK，现在的问题就是清理了，一般来讲真正占内存的文件一般就是那几个最大的，这里把最大的几个文件删除就好了。
首先通过finder的前往（Go）打开库文件夹（Library） 这里需要打开前往菜单的时候按住alt使library显示。
通过Action中sort by按大小排序，就可以找出所有占内存很大的文件了。

我这里发现前几天下载好多次之后不会用的doker还有android的idea编辑器都占了好多内存。。
果断删掉，于是多了30G内存
完成！
