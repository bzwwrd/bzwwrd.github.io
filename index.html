终端中打开文件

有时候终端用惯了就懒得切出去点图标了，记一下怎么用终端直接打开文件。
这里用sublime打开txt文件做例子

open -a /Applications/Sublime\ Text.app ~/Desktop/vpn.txt

当然每次打字这么一长串也很不舒服，我们可以写一个symbol link来简化它

ln -s "/Applications/Sublime Text.app/Contents/SharedSupport/bin/subl" /usr/local/bin/subl

这样以后只要输入subl + 文件绝对路径就可以啦

Mac清内存

前几天发现内存老不够用，今天索性清一下。

先点屏幕左上角About this Mac调出存储查询，看看哪个部分占内存最多。如果Document占的多，可以直接通过内置的management进行修改。这里主要介绍Others的清理方法。

首先必须明确Others里面都包含哪些内容。
所有的文档，包括 .pdf .doc .psd所有的压缩文件和磁盘映像，包括 .zip .dmg .iso所有的用户数据所有的系统文件，包括 System 文件夹 和 cache 文件夹所有的 Spotlight 不认识的文件，包括 .exe 在内的对 OS X 来说奇奇怪怪的后缀名所有的不是 OS X 标准格式的视频、音频文件所有的应用插件和扩展
大概就是各种平时用到的用不到的文件了，怪不得占内存这么多。。

OK，现在的问题就是清理了，一般来讲真正占内存的文件一般就是那几个最大的，这里把最大的几个文件删除就好了。
首先通过finder的前往（Go）打开库文件夹（Library） 这里需要打开前往菜单的时候按住alt使library显示。
通过Action中sort by按大小排序，就可以找出所有占内存很大的文件了。

我这里发现前几天下载好多次之后不会用的doker还有android的idea编辑器都占了好多内存。。
果断删掉，于是多了30G内存
完成！

人是属性的集合

最近少见的开始情绪化，看到不和自己想法的人或事情心里难免会有些抵触。

其实扪心自问长久以来，我的包容性确实是在慢慢变强的。甚至逐渐地接受了这个世界会有各种瑕疵。我记得小时候看到周围的孩子捡到钱装自己兜里心里都会抵触好久，现在怕是没有那种正义感了。

但说到底，人是属性的集合嘛。这个世界就是这么五花八门，有的人贪财，有的人好色，有的人追名，有的人逐利。好和坏只是主观的评价，我们看到的只是人所有属性的表现而已。前几天看综艺上李诞讲，一个人从小到大看到的都是他爸爸在客厅里挂的一张大船，等他十七八岁追女孩子买一个小帆船送给她，人家说丑，那这孩子也蛮委屈的。

天地不仁以万物为刍狗。老天不会因为你是个好人就奖励你。好是你的选择，勤奋是你为你的选择付出的努力。坏人如果勤奋，那老天也会给他应得的报酬。好没有错，错的是你没有坏人努力。

哈哈，到现在这个年龄在谈好坏总是显得有点孩子气，那就当孩子气是我的属性吧。
